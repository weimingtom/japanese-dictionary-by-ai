# 在线日语词典项目总结

## 项目概述

本项目是一个静态的在线日语词典网页应用，旨在为用户提供便捷的日语单词查询服务。该词典支持多种查询方式，包括按日语单词、读音或中文含义进行搜索，覆盖了日常交流中的常用词汇。项目采用纯前端技术栈实现，无需后端支持，可直接在浏览器中运行。

## 项目结构

- index.html: 主页面，包含搜索框、结果展示区和常用词汇列表
- css/style.css: 样式文件，定义了页面的整体外观和响应式布局
- js/dictionary.js: JavaScript文件，处理词典数据和搜索逻辑
- dictionary-data/: 词汇数据存储（在JS文件中定义）

## 功能特性

1. **智能搜索**: 支持按日语单词、读音或中文含义进行模糊搜索
2. **丰富词汇**: 包含超过400个常用日语词汇，涵盖日常生活各个方面
3. **快速查询**: 即时显示查询结果，支持键盘回车快捷搜索
4. **常用词汇**: 首页展示常用词汇列表，便于快速浏览
5. **响应式设计**: 适配不同屏幕尺寸，支持移动端访问
6. **直观界面**: 清晰的卡片式布局展示词汇信息

## 技术细节

- **前端技术**: HTML5, CSS3, JavaScript (ES6)
- **样式框架**: 纯CSS实现，无外部依赖
- **数据存储**: 词汇数据以JavaScript对象数组形式存储
- **搜索算法**: 使用filter方法实现客户端搜索
- **兼容性**: 支持现代浏览器，包括Chrome、Firefox、Safari等

## 使用方法

1. 打开index.html文件或部署到web服务器
2. 在搜索框中输入日语单词、读音或中文含义
3. 点击"查询"按钮或按回车键获取结果
4. 结果将以卡片形式展示日语单词、读音和中文含义

## 数据来源

词汇数据主要涵盖：
- 日常问候语（如：こんにちは、おはよう等）
- 数字表达（一至万的数字及其读音）
- 基础生活词汇（食物、交通、家庭、职业等）
- 常用动词形容词（基本动词和形容词变化）
- 时间日期表达（星期、月份、季节等）
